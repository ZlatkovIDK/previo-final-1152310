usuario 📊 Panel Admin

Endpoints disponibles

Usa Postman o cualquier cliente REST para probar los endpoints. Debes iniciar sesión primero (Spring Security maneja la sesión con cookies).

📨 Módulo 1 — Mensajes Internos

Mensajería interna entre usuarios registrados.

POST/api/mensajes
GET/api/mensajes/bandeja-entrada
GET/api/mensajes/enviados
PUT/api/mensajes/{id}/leer
GET/api/mensajes/no-leidos/count

📋 Módulo 2 — Solicitudes

Flujo de estados: PENDIENTE → APROBADA / RECHAZADA.

POST/api/solicitudes
GET/api/solicitudes/mis-solicitudes
GET/api/solicitudes ADMIN
PUT/api/solicitudes/{id}/aprobar ADMIN
PUT/api/solicitudes/{id}/rechazar ADMIN

📊 Módulo 3 — Panel Visual ADMIN

Visualización del estado de solicitudes con indicadores y tabla.

GET/admin/solicitudes/panel
→ Abrir Panel
